Итоговый результат на примере входного файла у меня получился stop 18. 

В примере в пдф-ки есть нестыковки с матрицей $T_{i,j}$:
1. Изделие k=3 на 1 станке должно закончиться в t=6 (0 + T1,1=6), а в примере написано 4
2. Изделие k=1 на Станке 0 должно закончиться в t=7 (3 + T1,0=4), и в примере опять таки написано 5

Также есть какие-то странности с порядком вывода событий в примере при совпадающем времени например при:
finish 4 3 1 1
ready 4 3 1
start 4 4 0 1

а указаный порядок finish -> start -> wait -> ready
